NOTABLE LESSONS FOR DOCS
  * The model accepted the async-IIFE pattern from the prompt without
    re-deriving it. This validates the documentation we added in the
    LAN-direct Network verification (top-level return / await are not
    allowed in QuickJS; wrap in an async IIFE) — once the recipe is in
    SKILL.md and visible to the model, success is single-shot.

  * Compared to the LAN-direct Network run that needed 4 syntax
    correction attempts before settling on the IIFE shape, supplying the
    correct shape in the prompt cuts wall-clock from ~13 s to ~12 s for
    the LLM-only portion (1 tool call vs. 5).
