WHAT THIS PROVES
  ✓ The LAN-direct UniFi Protect Integration API path
        https://<controller>/proxy/protect/integration/v1/...
    works against real Protect-enabled UniFi OS hardware with the same
    controller-issued local API key the Network surface uses (Network
    and Protect share auth on the local controller).
  ✓ The unifi.local.protect.* sandbox surface exposed by the QuickJS
    sandbox is wired correctly: it loads the official Protect OpenAPI
    spec from apidoc-cdn.ui.com (35 ops resolved for v7.0.107),
    dispatches both raw request() and named callOperation() lookups
    through the right /proxy/protect/integration prefix, and survives
    a multi-call traversal (2 host calls in 162 ms inside QuickJS).
  ✓ Per-tenant TLS handling honors UNIFI_LOCAL_INSECURE=true (the same
    flag that gates Network on the same controller) and emits the
    documented loud warning at [unifi.local.protect].
  ✓ The synthesizeOperationId function produces stable names for the
    official spec (which ships every operationId as null). The
    path-based request() escape hatch we used for portability also
    works cleanly.

WHAT THIS DOES NOT PROVE
  ✗ Mutation Protect operations (PTZ goto/patrol-start/patrol-stop,
    disable-mic, alarm-manager webhook trigger). This run is purely
    read-only: GET /v1/meta/info and GET /v1/cameras.
  ✗ Binary/streaming operations (snapshot bytes, RTSPS metadata,
    talkback, subscribe/* WebSockets) — all out of scope for the
    JSON-only HttpClient.
  ✗ End-to-end LLM-mediated invocation against the local Protect
    surface. Sandbox script driven directly from a Node script through
    ExecuteExecutor; a real client (cursor-agent, opencode, …) issuing
    the `execute` tool with the same code has not yet been exercised
    against local Protect.
  ✗ Multi-NVR / multi-tenant Protect setups. This run targets a single
    UDM-Pro Protect installation.

EXACT SANDBOX CODE EXECUTED
  var p = unifi.local.protect;
  var meta = p.request({ method: 'GET', path: '/v1/meta/info' });
  var cameras = [];
  try {
    var camRes = p.request({ method: 'GET', path: '/v1/cameras' });
    cameras = Array.isArray(camRes) ? camRes
            : (camRes && camRes.data) ? camRes.data : [];
  } catch (e) { cameras = ['ERROR: ' + String(e)]; }
  ({
    meta: meta,
    cameraCount: cameras.length,
    cameraSample: cameras.slice(0, 5).map(function(c) {
      return c && typeof c === 'object'
        ? {
            id: c.id, name: c.name, modelKey: c.modelKey, type: c.type,
            state: c.state, firmwareVersion: c.firmwareVersion,
            isMicEnabled: c.isMicEnabled, isRecording: c.isRecording,
            hasSpeaker: c.hasSpeaker, hasMic: c.hasMic
          }
        : { raw: c };
    })
  });
