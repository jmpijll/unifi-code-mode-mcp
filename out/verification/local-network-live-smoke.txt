        https://<controller>/proxy/network/integration/v1/...

    API key.
  ✓ The unifi.local.* sandbox surface (alias of unifi.local.network.*)
    exposed by the QuickJS sandbox is wired correctly: it loads the
    controller's matching OpenAPI spec from apidoc-cdn.ui.com (67 ops
    resolved for Network 10.1.84 — the closest published spec to the
    controller's 10.3.58), dispatches both raw request() and named
    callOperation() lookups through the right /proxy/network/integration
    prefix, and survives a multi-call traversal (10 host calls in
    608 ms inside QuickJS).
  ✓ Per-tenant TLS handling honors UNIFI_LOCAL_INSECURE=true on the
    self-signed UDM cert and emits the documented loud warning at
    [unifi.local].
  ✓ Both expected and unexpected per-operation errors propagate cleanly
    back into the sandbox as catchable JS Errors with the documented
    [unifi.local.http] HTTP <status> on <path>: <api-message> shape
    (verified via the firewallZones / firewallPolicies branches —
    Zone-Based Firewall is not enabled on this controller, so each
    raised HTTP 400 with code api.firewall.zone-based-firewall-not-
    configured; the sandbox script captured them into per-section
    *_error fields and continued the traversal cleanly rather than
    aborting).

WHAT THIS DOES NOT PROVE
  ✗ Mutation operations (create/update/delete) on the local Network
    surface — this run is purely read-only (overview/list endpoints
    only, no PATCH/POST/DELETE).
  ✗ End-to-end LLM-mediated invocation against the local surface. The
    sandbox script was driven directly from a Node script through
    ExecuteExecutor; a real client (cursor-agent, opencode, …) issuing
    the `execute` tool with the same code has been verified for the
    cloud paths but not yet for direct-local.
  ✗ Multi-site traversal — this controller has only the Default site,
    so per-site isolation in the sandbox loop is exercised but
    cross-site collisions are not.
  ✗ Behaviour under controller-side load — single low-traffic burst.

EXACT SANDBOX CODE EXECUTED
  var net = unifi.local;
  var snapshot = {
    generatedAt: new Date().toISOString(),
    info: net.request({ method: 'GET', path: '/v1/info' })
  };
  var sitesPage = net.callOperation('getSiteOverviewPage', { pageSize: 100 });
  snapshot.sites = [];

  var siteList = (sitesPage && sitesPage.data) || [];
  for (var i = 0; i < siteList.length; i++) {
    var s = siteList[i];
    var site = { id: s.id, name: s.name, internalReference: s.internalReference, raw: s };
    try { site.networks = net.callOperation('getNetworksOverviewPage',  { siteId: s.id, pageSize: 100 }).data; } catch (e) { site.networks_error  = String(e); }
    try { site.wans     = net.callOperation('getWansOverviewPage',      { siteId: s.id, pageSize: 100 }).data; } catch (e) { site.wans_error      = String(e); }
    try { site.wifi     = net.callOperation('getWifiBroadcastPage',     { siteId: s.id, pageSize: 100 }).data; } catch (e) { site.wifi_error      = String(e); }
    try { site.firewallZones    = net.callOperation('getFirewallZones',    { siteId: s.id }); } catch (e) { site.firewallZones_error    = String(e); }
    try { site.firewallPolicies = net.callOperation('getFirewallPolicies', { siteId: s.id }); } catch (e) { site.firewallPolicies_error = String(e); }
    try { site.aclRules         = net.callOperation('getAclRulePage',      { siteId: s.id, pageSize: 200 }).data; } catch (e) { site.aclRules_error = String(e); }
    site.devices = [];
    try {
      var devPage = net.callOperation('getAdoptedDeviceOverviewPage', { siteId: s.id, pageSize: 200 });
      var devs = (devPage && devPage.data) || [];
      for (var j = 0; j < devs.length; j++) site.devices.push({ summary: devs[j] });
    } catch (e) { site.devices_error = String(e); }
    try {
      var clientsPage = net.callOperation('getConnectedClientOverviewPage', { siteId: s.id, pageSize: 200 });
      site.clientsTotal  = clientsPage && clientsPage.totalCount;
      site.clientsSample = ((clientsPage && clientsPage.data) || []).slice(0, 5);
    } catch (e) { site.clients_error = String(e); }
    snapshot.sites.push(site);
  }
  snapshot;
