    config to `~/.claude.json`.
  ✓ `claude mcp list` performs a real protocol-level health check

CLAUDE CODE CLI AUTH CONSTRAINT (for testers)
  Claude Code CLI v2.0.47 in `--print` mode requires either:
    - `ANTHROPIC_API_KEY` set in the environment, OR
    - A prior `claude` interactive session that completed the
      Anthropic Console OAuth flow.
  Without one of those, `claude --print -p "..."` writes to
  `~/.claude.json` and then blocks indefinitely (no error, no
  progress log). This is documented behaviour of Claude Code CLI,
  not a bug in our server. Once a tester has Claude Code auth set
  up, this same MCP registration + the prompt template under
  `out/verification/cursor-agent-sonnet-mcp-call.txt` should drive
  the full end-to-end loop.
